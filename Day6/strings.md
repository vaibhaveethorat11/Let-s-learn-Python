## Strings:
Collection of text in python are called string and there special functions called string methods used to manipulate strings. It is one of the fundamental data types in python and used to represents text. 
It has abbreviated name in Python : str.

It has three important properties:
1. Contain individual letters or symbols called characters.
2. Have a length, defined as number of characters contain within string
3. Characters in a string appear in a sequence, each character has numbered position called index.

We can use single quotes or double quotes.
```py
str1 = "Hello"
str2 = 'Vaibhavee'
```
