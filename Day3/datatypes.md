Data Types in Python:
Data types define the kind of data a variable can hold and determine the operations that can be performed on it. In Python, every value is an object and each object belongs to a specific data type.
Python data types are given as below :
- Numeric
  - Integer
  - Complex Number
  - Float 
- Boolean
- Dictionary
- Set
- Sequence
  - String
  - Tuple
  - List
```py
a = "Hello World" # string
a = 20  # integer
a = 20.5  # float
a = 3i  # complex
a = ["red", "blue", "green"]  # list 
a = ("red", "blue", "green")  # tuple
a = {"name": "Vaibhavee", "age": 24} # dict
a = {"apple", "banana", "grapes"} # set
a = True  # bool
```
